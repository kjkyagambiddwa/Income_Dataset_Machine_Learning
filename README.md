Classifying Income Level: A Statistical Analysis of the Adult Census Dataset
This repository contains a comprehensive statistical study and machine learning implementation focused on predicting whether an individual's annual income exceeds $50,000 based on census data. The project involves rigorous data preprocessing, feature engineering, and a comparative analysis of multiple classification algorithms.

🚀 Project Overview
The objective of this analysis is to identify the primary socio-economic and demographic drivers of income levels. By leveraging the Adult Census Dataset, this study evaluates various modeling techniques to determine the most effective approach for binary income classification.

🛠️ Methodology & Workflow
Data Preprocessing: Handling duplicates, analyzing missing values, and implementing KNN Imputation for data integrity.

Outlier Treatment: Detailed stage-wise outlier detection and targeted treatment to ensure model robustness.

Exploratory Data Analysis (EDA): Class distribution analysis, correlation matrices, and categorical variable associations.

Feature Selection: Creating specialized datasets tailored for specific algorithms (e.g., Decision Trees vs. Naive Bayes).

Modeling: Implementation of multiple supervised learning models:

Decision Trees (Standard and Balanced)

Naive Bayes

Artificial Neural Networks (ANN) (Standard, Balanced, and Tuned)

Logistic Regression

K-Nearest Neighbors (KNN) with K-optimization

Evaluation: Comparative analysis using Accuracy, Sensitivity, Specificity, and ROC/AUC metrics.

📊 Key Findings
Identification of top predictors (Odds Ratios) through Logistic Regression.

Visual decision rules derived from Balanced Decision Trees.

Comprehensive performance summary across all five major model types.
